## 📌 Overview

CopyCraft AI is a next-generation Generative AI marketing platform that transforms raw product descriptions into professional, platform-specific marketing content. Built using Large Volume Models (LLMs), advanced prompt engineering, and intelligent content optimization techniques, the platform enables marketers, startups, creators, and businesses to generate compelling copy within seconds.

The application supports multiple platforms, audience personas, brand voices, content tones, SEO optimization, campaign generation, and AI-powered quality scoring through a modern SaaS-inspired interface.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/CopyCraft-AI.git

cd CopyCraft-AI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure API Key

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key_here
```

or

```env
OPENAI_API_KEY=your_api_key_here
```

---

## ▶️ Run Application

```bash
streamlit run app.py
```

Application will launch locally at:

```text
http://localhost:8501
```
